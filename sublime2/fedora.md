Fedora 18, Sublime2



Adicione o repostitório
	
	# yum-config-manager --add-repo http://repo.cloudhike.com/sublime2/fedora/sublime2.repo

Habilite po repostitório

	# yum-config-manager --enable

Instale
	
	# yum install sublime-text
