FEDORA 18+


Execute o seguinte comando:

	# yum install samba samba-client samba-doc samba-swat

Inicialize o serviço (atalho)

	# service smb start

Para iniciar o serviço:

	# /etc/init.d/samba start

Para parar o serviço:

	# /etc/init.d/samba stop
